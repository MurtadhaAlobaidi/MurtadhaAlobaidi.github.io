Music School Website

Project Overview

Developed a console-based website for a music school, focusing on an intuitive user interface and efficient database interaction. The platform is designed to handle various administrative tasks related to a music school's operations.

Technical Details

Built using Java and JDBC in an MVC architecture, ensuring a structured and maintainable codebase. Data is managed through PostgreSQL databases, with tools such as pgAdmin, DBeaver, GitHub, and Azure Data Studio employed for database interaction and version control. The project involved SQL querying and package management using npm.

Project Results

The completed website offers robust data handling capabilities, significantly streamlining the administrative processes of the music school. It has been commended for its user-friendly console interface and effective data management system.
