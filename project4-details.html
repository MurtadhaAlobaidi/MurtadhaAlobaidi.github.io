Morse Mate

Project Overview

Developed "Morse Mate," a unique web application that enables users to translate text into Morse code and transmit it to Arduino hardware for audible playback. This project focuses on integrating web technology with hardware to create an interactive Morse code experience.

Technical Details

The website was built using a combination of C++, Vue.js for the frontend, and Bootstrap for styling. Backend integration was achieved with JavaScript and Firebase for database storage. Version control was managed via GitHub. The project also involved interfacing with Arduino and ESP8266 WiFi modules for hardware interaction.

Project Results

Morse Mate successfully bridged the gap between digital text and physical Morse code transmission, offering users an innovative way to interact with Morse code. The website was praised for its seamless integration of web technologies with hardware components, providing a unique and educational experience.
